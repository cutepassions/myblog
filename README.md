# myblog
## 1. 개인 블로그 구축
     
프로젝트 개요 : 반응형 웹과 기타 기능들을 통한 블로그 구축   

사용 스킬 : HTML5, CSS, javascript, PHP, mariaDB

대표적인 기능
- 회원가입
- 로그인/로그아웃
- 회원정보 변경 및 탈퇴
- 쪽지
- 게시판 (수정, 삭제, 검색, 파일 업로드)   

소요기간 : 3개월   
프로젝트 결과 : 일반적인 블로그 형식의 웹 사이트 구축   
완성된 사이트 : http://woosuk.izerone.co.kr:8090/~s120171838/   

참고 자료 : https://www.w3schools.com/
